Формат вводимых данных:
На первой строке располагается название базы знаний.
Затем через знак перевода строки - список вопросов.
После вопросов - следует список событий (вариантов) в формате:
Событие<Перевод строки>Начальная_вероятность_события номер_вопроса) вероятность_max вероятность_min<перевод строки>
Пример ввода:
Определение пола


Вы мальчик?
Вы девочка?

Мальчик
0.5 1) 1 0 2) 0 1
Девочка
0.5 2) 1 0 1) 0 1
Заметьте, номера вопросов можно устанавливать не по порядку + вопросы не влияющие на вероятность события можно опускать.
